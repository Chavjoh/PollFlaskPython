PollFlaskPython
===============

Python implementation with Flask micro-framework of a spread poll.

## Requirements

* Flask 0.10.1
* Jinja2 2.7.2
* MarkupSafe 0.19
* MySQL-Connector-Python 1.1.4
* Werkzeug 0.9.4
* itsdangerous 0.24
* pycrypto 2.6
* pygame 1.9.2a0
* pywin32 218
* virtualenv 1.11.4
